# Sales Data Analysis Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a global sales dataset using Python.

The goal is to analyze revenue, profitability, regional performance, and sales channel effectiveness to extract meaningful business insights.


## Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook


##  Key Business Insights

- Sub-Saharan Africa generates the highest total revenue.
- Cosmetics is the most profitable product category.
- Offline sales channel generates higher profit compared to Online.
- North America contributes the least revenue and may require strategic focus.



## Visualization
A bar chart was created to analyze revenue distribution by region.



##  Conclusion
This project demonstrates fundamental data analysis skills including data cleaning, aggregation, statistical analysis, visualization, and business interpretation.
